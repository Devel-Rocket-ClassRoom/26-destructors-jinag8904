# 도서관 좌석 관리

도서관 좌석 이용을 관리하는 시스템을 구현하시오.

`Seat` 클래스를 작성하여 다음 기능을 구현:

1. 정적 필드로 총 이용 횟수와 현재 착석 인원을 추적
2. 각 좌석은 고유한 ID를 가짐 (1부터 시작하여 순차 증가)
3. 생성자에서 학생 이름을 받고 "좌석 {id}번 착석: {학생이름}" 출력
4. 소멸자에서 "좌석 {id}번 반납: {학생이름}" 출력
5. `Study()` 메서드로 "{학생이름}이(가) 좌석 {id}번에서 공부 중..." 출력
6. `ShowStatus()` 정적 메서드로 현재 상태 출력

## 예상 실행 결과

```
좌석 1번 착석: 김민수
좌석 2번 착석: 이지영
좌석 3번 착석: 박서준
김민수이(가) 좌석 1번에서 공부 중...
이지영이(가) 좌석 2번에서 공부 중...
박서준이(가) 좌석 3번에서 공부 중...
총 이용: 3명, 현재 착석: 3명
좌석 3번 반납: 박서준
좌석 2번 반납: 이지영
좌석 1번 반납: 김민수

총 이용: 3명, 현재 착석: 0명
```

> **참고**: 소멸자 호출 순서는 GC에 의해 결정되므로, 
반납 순서는 위 예시와 다를 수 있습니다.
